## 一、项目亮点（Project Highlights）

- ✅ **100% 支持 俱乐部玩法**  
  - 支持 6~9 人俱乐部桌，自动配桌、自动结算  
  - 支持多桌并行、多盲注、多底池结构  
- ✅ **俱乐部与代理解体系**  
  - 俱乐部创建、成员管理、俱乐部专属房间与赛事  
  - 代理解/分润模型，支持多层级代理解结构  
- ✅ **安全与风控**  
  - 内置防作弊结构，可对接风控系统，支持反串通、反机器人、反多开  
- ✅ **可扩展性**  
  - 模块化设计，可支持 SNG / MTT、短牌、AOF、德州牛仔等新玩法扩展  
- ✅ **多端支持**  
  - 可为 Web(H5)、App、小程序、社交平台提供统一后端与 API 接口
